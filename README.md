# Premier League Match Predictor

This app predicts Premier League match outcomes using machine learning and rolling team statistics. It uses data from the 2021–2022 seasons and calculates rolling stats based on each team's last 3 games. Predictions are experimental and for entertainment and analysis only. The app does not promote betting and is not liable for any betting decisions.

## Features

- User-friendly interface
- Rolling stats for team performance
- Machine learning-based predictions

## Data

Historical match data from the 2021–2022 Premier League seasons.

## Disclaimer

Predictions may not reflect current season performance. Use for entertainment and analysis only.
